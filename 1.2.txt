// Задание: Что выводится в консоль и почему?
// const a = () => console.log("a");
// const b = () => console.log("b");
// константа с = а || б ? а() : б();
// console.log(с);

Вывод будет таким: "a" и "undefined".
Объяснение: При выполнении выражения "a||b" возникает ли функция "a" истинным значением. В этом случае "a" является истинным значением, поэтому выводится "a". После вызова a() функция "a" не фигурирует в значении "undefined", поэтому "c" принимает значение "undefined".
